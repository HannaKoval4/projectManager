## 8. Схема взаимодействия компонентов (MVVM)

```
┌──────────────┐         ┌──────────────┐         ┌──────────────┐
│     VIEW     │◄────────┤   VIEWMODEL  │────────►│    MODEL     │
│   (XAML)     │  Binding│  (Логика)    │  Uses   │   (Данные)   │
└──────────────┘         └──────┬───────┘         └──────┬───────┘
                                 │                         │
                                 │                         │
                                 ▼                         ▼
                          ┌──────────────┐         ┌──────────────┐
                          │   SERVICE     │         │   DATABASE   │
                          │ (DatabaseService)        │  (SQL Server) │
                          └──────────────┘         └──────────────┘
```

## 10. Компоненты приложения (детально)

### Слой представления (Views)
```
Views/
├── LoginWindow.xaml          - Окно авторизации

├── ProjectsView.xaml          - Представление проектов
├── TasksView.xaml             - Представление задач
├── TeamView.xaml              - Представление команды
├── StatisticsView.xaml        - Представление статистики
├── ProjectDialog.xaml         - Диалог проекта
├── TaskDialog.xaml            - Диалог задачи
└── EmployeeDialog.xaml        - Диалог сотрудника
```

### Слой бизнес-логики (ViewModels)
```
ViewModels/
├── MainViewModel.cs           - Управление навигацией

├── TasksViewModel.cs          - Логика задач
├── TeamViewModel.cs           - Логика команды
├── StatisticsViewModel.cs     - Логика статистики
├── ProjectDialogViewModel.cs  - Логика диалога проекта
```

```
Models/
├── Project.cs                 - Модель проекта
```

```
Data/

Services/
```

### Конвертеры

```
Converters/
```
