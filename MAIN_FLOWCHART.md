## Описание основных блоков

### 1. ИНИЦИАЛИЗАЦИЯ
- Запуск приложения через App.xaml
- Открытие окна авторизации
- Проверка подключения к БД

### 2. АВТОРИЗАЦИЯ
- Ввод логина и пароля
- Валидация полей
- Переход к главному окну

### 3. ГЛАВНОЕ ОКНО
- Инициализация MainViewModel
- Загрузка данных из БД
- Отображение раздела "Проекты" по умолчанию

### 4. РАЗДЕЛЫ ПРИЛОЖЕНИЯ
- **Проекты**: CRUD операции с проектами
- **Задачи**: Управление задачами + фильтрация
- **Команда**: Управление сотрудниками
- **Статистика**: Аналитика и метрики

### 5. ОПЕРАЦИИ С ДАННЫМИ
- Валидация входных данных
- Транзакционная обработка через DatabaseService
- Сохранение в MS SQL Server
- Обновление View через PropertyChanged

### 6. ОБРАБОТКА ОШИБОК
- Перехват исключений
- Откат транзакций при ошибках
- Информирование пользователя

### 7. ЗАВЕРШЕНИЕ
- Закрытие соединений с БД
- Закрытие окон
- Завершение работы приложения

## Ключевые потоки данных

1. **Поток создания проекта/задачи:**
   View → ViewModel → Validation → DatabaseService → DbContext → SQL Server → Update View

2. **Поток фильтрации:**
   Filter Change → ViewModel.ApplyFilters() → LINQ Query → Filtered Collection → Update View

3. **Поток навигации:**
   Navigation Button → NavigateCommand → MainViewModel.Navigate() → UpdateView() → Load New View

4. **Поток транзакции:**
   Operation → BeginTransaction → Execute → SaveChanges → Commit/Rollback → Update View
